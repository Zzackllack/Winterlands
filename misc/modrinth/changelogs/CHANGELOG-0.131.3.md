# Winterlands 0.131.3 (Alpha)

- First Modrinth alpha publish for NeoForge 21.1.215 (MC 1.21.1).
- Features:
  - Festive ambience: holiday lighting, snow-forward visuals, and seasonal decor.
  - Custom worldgen to keep biomes wintry and scenic.
  - Performance layer: client optimizations and QoL to keep performance smooth.
  - Custom Winterlands main menu via FancyMenu.
- Please report bugs and balance issues on the GitHub tracker: https://github.com/Zzackllack/Winterlands/issues.
